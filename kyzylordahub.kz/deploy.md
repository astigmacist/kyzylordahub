# Деплой на ps.kz хостинг

## Шаг 1 — Собрать проект

```bash
cd kyzylordahub.kz
npm install
npm run build
```

После билда появится папка `out/` — это и есть статический сайт.

## Шаг 2 — Что загружать на хостинг

Загрузить **содержимое папки `out/`** в корень сайта на ps.kz (обычно папка `public_html` или `www`).

Структура в `out/` будет такой:
```
out/
├── index.html          ← главная страница (ps.kz найдёт её автоматически)
├── _next/              ← JS, CSS, статика Next.js
│   └── static/...
├── .htaccess           ← Apache конфиг (копируется из public/)
└── ... прочие файлы
```

## Шаг 3 — Загрузка через FTP/FileManager

1. Войдите в панель управления ps.kz
2. Откройте **Файловый менеджер** (или подключитесь через FTP)
3. Перейдите в папку `public_html` (или `www`)
4. Загрузите **все файлы из папки `out/`** (не саму папку, а её содержимое!)
5. Убедитесь что файл `.htaccess` загрузился (он скрытый)

## Шаг 4 — Проверка

Откройте ваш домен — должна открыться главная страница.

## ⚠️ Важно

- Видеофайл `kyzylorda-hub.mp4` (~247 МБ) — ps.kz может иметь ограничение на размер файлов при загрузке.
  Если не загружается через файловый менеджер — используйте FTP-клиент (FileZilla).
- `.htaccess` файл хранится в папке `public/` проекта и автоматически копируется в `out/` при билде.
